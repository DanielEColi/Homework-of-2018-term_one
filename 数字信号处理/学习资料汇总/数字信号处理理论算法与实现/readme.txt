一、光盘内容

    光盘中含有《数字信号处理－理论、算法与实现》一书中所涉及到的绝大部分算法。

    \Dsp_c		C语言编写的算法程序。
    \Dsp_fortran	FORTRAN语言编写的算法程序。
    \Dsp_matlab		用MATLAB编写的信号处理程序，包含本书各个章节的
                        大部分例题。

二、运行环境

    硬件环境：Pentium 200以上计算机，64M内存，真彩色显示卡。
    软件环境：Windows 98/NT/2000，Turbo C，FORTRAN77 V5.10，
              MATLAB 5.X以上版本；
